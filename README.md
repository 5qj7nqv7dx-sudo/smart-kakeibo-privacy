プライバシーポリシー（Privacy Policy）

SmartKakeibo（以下、「本アプリ」） は、ユーザーのプライバシーを尊重し、以下のとおり個人情報の取り扱いについて定めます。

⸻

1. 収集する情報について

本アプリでは、ユーザー個人を特定できる情報（氏名、住所、メールアドレス等）を 直接収集することはありません。

ただし、以下の情報が 第三者サービスを通じて自動的に収集される場合があります。

⸻

2. 広告配信について

本アプリでは、第三者配信の広告サービスとして Google AdMob を使用しています。

AdMob は、広告配信のために以下の情報を収集する場合があります。
	•	デバイス情報（端末の種類、OS など）
	•	広告識別子（IDFA 等）
	•	アプリの利用状況

これらの情報は、広告の表示およびパーソナライズ広告の提供を目的として使用されます。

Google 広告に関する詳細

Google の広告におけるデータの取り扱いについては、以下をご確認ください。
https://policies.google.com/privacy?hl=ja

⸻

3. 利用目的

収集された情報は、以下の目的にのみ利用されます。
	•	広告の表示および最適化
	•	アプリの品質改善・不具合修正

⸻

4. 個人情報の第三者提供について

本アプリが取得する情報は、法令に基づく場合を除き、第三者に提供されることはありません。

⸻

5. データの管理について

本アプリ内で入力された家計簿データは、すべて端末内に保存され、外部サーバーへ送信されることはありません。

⸻

6. プライバシーポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。
変更後のプライバシーポリシーは、本アプリ内または公開ページにて告知します。

⸻
